# Flask Docker App

Простое Flask приложение, упакованное в Docker.

## Функциональность

- Главная страница возвращает "Hello, User!"
- Приложение запускается на порту 5000
- Доступно через порт 5001 на хосте

## Запуск

```bash
docker-compose up --build
